API Integration Docs
Integration Guide

Integrate the ThinkDome sandbox directly with your custom agents or python scripts using HTTP requests. Replace headers with your generated API Key.

1. Python Request Example (httpx)
import httpx

api_key = "gsk_YOUR_GENERATED_API_KEY"
url = "http://localhost:8000/v1/orchestrate"

payload = {
    "type": "tool_use",
    "id": "toolu_01Hh5Ii6Jj7Kk8Ll9Mm0Nn",
    "name": "run_code",
    "input": {
        "language": "python",
        "code": "print('Hello from External LLM!')"
    }
}

headers = {
    "Authorization": f"Bearer {api_key}",
    "Content-Type": "application/json"
}

response = httpx.post(url, json=payload, headers=headers)
print(response.json())
2. cURL CLI Example
curl -X POST http://localhost:8000/v1/orchestrate \
  -H "Authorization: Bearer gsk_YOUR_GENERATED_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "tool_use",
    "id": "toolu_01Hh5Ii6Jj7Kk8Ll9Mm0Nn",
    "name": "read_file",
    "input": { "path": "README.md" }
  }'
